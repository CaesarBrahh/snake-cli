# Terminal Snake (C + ncurses)

A classic Snake game implemented in C using the ncurses library for real-time terminal rendering and input handling.

## Controls

Key 	Action

↑ 		Move Up
↓		Move Down
←		Move Left
→		Move Right


---

## How to Run

1. Install ncurses

macOS (Homebrew)
brew install ncurses

Ubuntu / Debian
sudo apt install libncurses-dev

2. Build
gcc main.c -lncurses

3. Run
./a.out
